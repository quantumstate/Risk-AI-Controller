Map Format

The map is represented in json with the structure: 
{"image": "map.png",
 "countries": [{"color": [133, 133, 255], 
                "neighbours": [23, 32, 37, 7],
                "continent": 4,
                 "centre": [268, 309]},
               ...]
 "continentVals":[5,2,...]
}

Default Continent Colors

1:red 2:yellow 3:green 4:cyan 5:blue 6:magenta 7:grey
